WHAT IS AR?

Augmented Reality (AR) is a view of the environment of the real world, augmented with computer generated graphics, sound, video, or GPS data which is usually seen through the lens of a camera. AR is implemented for many purposes such as marketing, educational, and recreational. Future potential usage of AR is yet to be improved. Unlike VR, which is a closed service that confines a user in a virtual space that can be used only in a limited space, AR can be used on the move. In a word, if VR is writing a console gaming device on the face, it is said that AR is like using a transparent mobile phone.
